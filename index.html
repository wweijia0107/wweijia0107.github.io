About

Building reliable geospatial AI for environmental decision-making.

My research connects GeoAI, Earth observation, and AI safety. I am especially interested in evaluating when remote-sensing vision-language models fail, how these failures vary across geographic regions and land-cover types, and how model behavior can be made more transparent for scientific and societal use.

My recent work explores instruction-tuned vision-language models for multi-label land-cover classification, parameter-efficient fine-tuning with LoRA, model steering, and evaluation under domain shift. Going forward, I aim to develop auditing frameworks for safe and responsible geospatial foundation models.

Projects

Research and technical projects

Remote-sensing VLM adaptation for land-cover tagging

Fine-tuned an instruction-tuned vision-language model on BigEarthNet-S2 for multi-label land-cover classification using parameter-efficient LoRA adaptation. The project emphasizes reproducible evaluation, steering behavior, and failure analysis.

  • Instruction-tuned VLMs
  • LoRA fine-tuning
  • Multi-label classification
  • BigEarthNet-S2

Ground-Penetrating Radar inversion

Designed a TransUNet inversion framework for the Kaggle gprMax Deep Learning Challenge, reconstructing 2-D subsurface permittivity maps from multi-offset GPR B-scan data with physics-aware augmentation and composite loss design.

  • TransUNet
  • Vision Transformer
  • Physics-aware loss
  • IoU 0.873

Remote-sensing LLM prototype

Built and deployed a domain-adapted VisualGLM-6B system for natural-language queries over Sentinel-2 imagery and Google Earth Engine assets, using LoRA fine-tuning and multimodal prompt fusion for environmental change-detection workflows.

  • VisualGLM-6B
  • GeoJSON tokenizer
  • Gradio dashboard
  • Triton deployment
